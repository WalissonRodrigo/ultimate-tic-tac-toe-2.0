Uma reinterpretação moderna, estratégica e futurista do clássico Jogo da Velha. O **Ultimate Tic-Tac-Toe 2.0** eleva o desafio ao introduzir uma grade global composta por 9 sub-grades locais, onde cada jogada define o território válido para o próximo turno.

![Ultimate Tic-Tac-Toe Preview](/assets/favicon.png)

## 🚀 Demonstração

Jogue agora: [https://walissonrodrigo.github.io/ultimate-tic-tac-toe-2.0/](https://walissonrodrigo.github.io/ultimate-tic-tac-toe-2.0/)

## ✨ Funcionalidades Principais

- 🧠 **Estratégia Meta-Grid**: Ganhe 3 sub-tabuleiros em linha para conquistar a vitória global.
- 🤖 **IA Adaptativa**: Desafie uma Unidade IA com lógica de antecipação e bloqueio.
- 🌐 **Deep Localization (i18n)**: Suporte completo para **Português (BR)**, **English (UK)** e **Español**, incluindo SEO e branding dinâmico.
- 📱 **PWA (Progressive Web App)**: Instale o jogo no seu celular ou desktop e jogue offline.
- 💾 **Persistência Local**: Progresso da partida, codinome e placar global salvos automaticamente no `localStorage`.
- ⚡ **Performance 100/100**: Otimizado para o Google Lighthouse em Performance, Acessibilidade, Boas Práticas e SEO.

## 🛠️ Tecnologias Utilizadas

- **Core**: HTML5 Semântico, Vanilla JavaScript (ES6+).
- **Style**: CSS3 Custom Properties, Glassmorphism, Neon Glow FX, Grid & Flexbox.
- **PWA**: Service Workers & Web App Manifest.
- **Icons**: SVG Inline com filtros de brilho dinâmicos.
- **Fonts**: Google Fonts (Orbitron & Rajdhani).

## ♿ Acessibilidade

O projeto foi construído seguindo as diretrizes de acessibilidade web:

- Navegação via landmarks (`<main>`, `<aside>`).
- Labels explícitas e atributos ARIA para leitores de tela.
- Contraste otimizado para a estética Cyberpunk.

## 📦 Como rodar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/WalissonRodrigo/ultimate-tic-tac-toe-2.0.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. (Opcional) Use uma extensão como _Live Server_ para testar as funcionalidades do Service Worker.
